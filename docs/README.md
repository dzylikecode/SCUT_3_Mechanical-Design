## 内容结构(七部分)

- 总论
- 常用机械零件的类型和选择

- 摩擦学设计
- 热分析
- 结构设计

## 本书特点

- 按学科分类编写

  淡化对具体机械零件的介绍

  突出各零件设计中的共性，便于建立不同零件在设计方法上的联系

- 加强对机械零、部件结构设计内容的介绍

- 增加当前机械学科的新技术、新工艺、新材料方面的内容

- 增加设计方法方面的内容

  特别是强调多方案设计、优化设计和再设计的思想及其在工程设计中的应用

书中在目录后，增加了符号说明；书后附录包含了各章常用的表格、线图和机械设计常用名词中英文对照。
