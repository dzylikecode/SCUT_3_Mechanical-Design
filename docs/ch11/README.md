# 机械零件摩擦设计

## 带传动设计

- 紧边和松边
- 应用于高速级传动

- 最大有效拉力

  $$F_{ec} = 2F_0 \frac {1 - \frac {1} {e ^ {f \alpha}}} {1 + \frac {1} {e ^ {f \alpha}}}$$

  $F_0$: 预紧力, 正相关 => 过大, 发热, 磨损

  $\alpha$: 包角 正相关 => 与 i 和 a 有关

  $f$: 摩擦系数 正相关

  - 打滑
